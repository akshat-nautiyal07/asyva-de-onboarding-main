# 02 PL/SQL / Procedural SQL

## Objective
Be able to read, understand, and write basic stored procedures and error handling.

## Assignment
### Tasks
1. Write one procedure that loads data into a target table (insert/update).
2. Add error handling and meaningful messages/logging.
3. Read a procedure (provided by mentor or online sample) and explain control flow in your own words.

### Learning links (official)
- Oracle PL/SQL docs (intro): https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/


## Expected outputs (acceptance criteria)
- `procedures.sql` (procedure with params; insert/update)
- `error_handling.sql` (exception handling pattern; meaningful message)
- Procedure code is commented and readable


## Mentor review checklist
- Can you explain the flow line-by-line?
- Can you tell when a stored procedure is justified vs plain SQL?
