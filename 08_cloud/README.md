# 08 Cloud Basics (AWS + Azure)

## Objective
Understand the cloud primitives used in ASYVA projects (storage, identity, secrets, compute).

## Assignment
### Tasks
AWS
- Explain S3 bucket/prefix/object layout
- Explain IAM roles/policies (concept)
- Explain how data moves from S3 to warehouse/lakehouse

Azure
- Explain ADLS Gen2 (account/container/folder)
- Explain Azure Databricks role
- Explain Key Vault usage

Security
- Write basic do/don't rules for secrets
