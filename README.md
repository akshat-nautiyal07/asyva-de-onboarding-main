# ASYVA Data Engineering Onboarding (Fresher Track)

## Goal (2 weeks, 8 hrs/day)
By the end of this bootcamp you should be able to:
- Write production-ready SQL (joins, CTEs, window functions, dedup patterns)
- Write clean Python ETL scripts with logging and error handling
- Build a Bronze → Silver → Gold pipeline in PySpark (Databricks CE OK)
- Understand basic data modeling (fact/dim, grain, SCD)
- Execute basic Snowflake loading (stages, COPY INTO, time travel)
- Explain AWS S3/IAM basics and Azure ADLS/Databricks/Key Vault basics

## How to use this repo
- Each folder has a README with:
  - Objective
  - Assignment
  - Expected outputs (acceptance criteria)
- Commit work daily.
- Keep code runnable and documented.

## Suggested workflow
1. Create a branch per day: `day-01-foundations`, `day-02-sql-basics`, ...
2. Commit at least once per day.
3. Open a PR for mentor review.

## Deliverables checklist (end of bootcamp)
- Completed modules 00 → 08
- Final project in `final_project/`
- Clear READMEs and runnable code

## Git requirement
- Complete `09_git/` early (Day 0/Day 1) so you can work via branches + PRs.
