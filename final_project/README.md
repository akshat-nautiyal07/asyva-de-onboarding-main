# Final Project

## Objective
Build and explain an end-to-end pipeline and analytics layer.

## Assignment
### Tasks
1. Choose a dataset (orders/payments or public dataset).
2. Build Bronze→Silver→Gold in PySpark.
3. Persist outputs (Parquet).
4. Write SQL queries for analytics (KPIs).
5. Provide architecture diagram and clear README.

### Requirements
- Re-runnable (idempotent) runs
- Logging + clear errors
- Clean folder structure


## Expected outputs (acceptance criteria)
- `architecture.png`
- `pipeline/` (code)
- `sql/` (analytics queries)
- `README.md` (run steps + design decisions + trade-offs)


## Mentor review checklist
- Can you explain the system without opening code?
- Can you debug a failure introduced by mentor?
