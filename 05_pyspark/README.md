* Pysparks let us to use python to work with big data.
* Data is generally considerd big data when i exceeds the processing capacity of a standard realtional databas, which typically struggles once dataset reach the Terabyte (TB) to PetaByte(PB).
* Apache sparks run data processing accresss many machines or clusters.

* A cluster is a group of computers that work together as one system to process data
      * Driver Node -> Control the job
      * Worker Node -> do the actual data processing

# Why we need clusters
  * To handle large data
  * Faster Processing
  * Parallel Processing
